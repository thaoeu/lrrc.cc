Firefox 全屏启动

2021年2月27日

本 文记录了我对 Firefox 启动项的一些尝试。

最开始我的目的非常简单，就是像 Chrome 那样只在新标签也显示书签栏，经过各种检索我找到了方法，虽然很麻烦，但是，能用。

不过后来一次更新后这个方法失效了，我不晓得为什么这样方便的功能火狐不能跟进，不过现在我找了解决方法： ，恰巧图标还是蓝色的，我 喜欢。

后来打算弄成类似！Vivaldi 文中那样，隐藏标签栏及搜索框的样子，有是一阵折腾，效果非常不好，再加上有段时间 Vivaldi 账号时常掉线，索性不用了。

退而求其次，我想让 Firefox 全屏运行，检索到的方法让我以 Kiosk 模式启动，这个模式可太坑爹了，它设计的初衷是给公共电脑使用，类似图书馆这种场所，给你一个不能退出的单浏览器模式，这也就意味着这种模式强制全屏，甚至连搜索框都无法调出，果断放弃。

最后还是不得以多装了个插件，我本意是不想多装任何一个插件的，连好用的油猴脚本我的单独放在另一个浏览器里，就是为了尽可能减少浏览器资源占用，防止其变的臃肿，但唯独插件给了我最舒服的体验，不得不说这也是一种取舍。
